FAQ

1.What is docjini?

Docjini is a Document Search Engine , which can be used to search Open Source Documents across the web. The term open Source Documents mean that which are publicly available on the internet

2.How can I find various files and documents online?

You can use Docjini to search various documents across the web by following the below steps:

  1. Visit docjini's homepage.
  2. Then Enter the search term or keyword of the document which you want to search.
  3. Then Choose the file format , i.e, in which format you want the documents.
  4. Click Search
That's all after you click search docjini will show all the matching documents across the web.
